# 人物圣经

## 盘总

| 维度 | 内容 |
|---|---|
| 外貌/年龄 | 30岁。高瘦+薄肌，定制衬衫袖子永远卷到肘部（不自知的习惯性动作）。常年把玩同一枚打火机（不抽烟）。眼神是园区最安静的眼神，安静到让人不舒服。 |
| 深层欲望 | 被允许成为一个普通人。不是逃跑不是成功，是有一天可以在一个不认识他的城市开一家不重要的小生意。 |
| 核心恐惧 | 白活——他最终什么都没有选择，只是被系统运行到终点。 |
| 羞耻点 | 他是养子。园区是别人给他的不是他建的。所有强控制欲背后：如果从头来过没有这个园区他是谁？不知道答案让他羞耻。 |
| 情感债务 | 对那些他本可以放走但没有放走的人。他知道园区是绞肉机，他选择继续运转。他记着每一个具体的人。 |
| 内在矛盾 | 绝对理性与深处孤独的撕裂。能计算一切，但计算不了为什么在女主提过的那个城市名下登记了一套房。 |
| 弧线方向 | 从"系统的执行者"走向"第一次为自己做一个决定"。无论哪条结局线，终点都是某种自主选择——只是有的选择是死，有的是交出自己，有的是帮她离开。 |
| 语音签名 | 说话慢，句子短，不用反问句。永远不解释决定只陈述结果。极少数真实时刻出现奇怪语气——不是温柔是比温柔更底层的东西。"那个城市，我去过一次。""知道。""没关系。"——他的"没关系"从来不是安慰对方，是他在告诉自己。 |
| 压力反应 | 收缩，变安静，踱步后坐下，倒酒不喝。危险越大越不动，在算，算完发现没有好出路就选最体面的坏结局。发现女主卧底时没有愤怒——可能很早就开始怀疑了只是不去确认。烧报告不是冲动是想清楚的决定。本质是一种提前的告别。 |
| 前史 | 生母不详，缅甸华人黑帮家族收养。养父对他好是"培养接班人"的好不是"养一个孩子"的好。12岁开始学账目学谈判学看人眼睛判断有没有说谎。18岁养父死他接手，清掉书桌上所有东西换成自己的。第二天坐在椅子上跟什么都没发生一样。 |

## 女主

| 维度 | 内容 |
|---|---|
| 外貌/年龄 | 28岁。清秀，人群里不是最先被注意到的。单独相处有专注感让对方觉得被认真对待——接近工具也是真的，分不清边界。穿着中性功能性强。 |
| 深层欲望 | 知道她的感受属于她自己不属于任务。 |
| 核心恐惧 | 任务结束后不知道"她自己"是什么了。 |
| 着耻点 | 她曾在某一刻真的希望任务失败。没有告诉任何人包括上线。 |
| 情感债务 | 对每条线里靠近的那个人。她用真实情感完成任务但无法归还——不属于档案但也不知道属于哪里。 |
| 内在矛盾 | 被训练成功能但想知道自己作为一个人是什么。任务成功和这个问题的答案有时同向有时背道。 |
| 弧线方向 | 从"情感是工具"走向"第一次允许自己不知道这个感受算什么"。每条线终点都没完全解决但有的她停下来了有的继续走了更远更不确定。 |
| 语音签名 | 问问题比陈述多。问题不是收集信息是制造空间——让对方多待一会儿多说一点。偶尔重复对方最后一句话——让对方知道她真的听见了。自己说话句子干净没有修饰词，和工作时"温柔专注"形成落差，注意到落差的人是真的在看她。 |
| 压力反应 | 多余确认动作：检查已确认信息、走回头检查关好的门、报告写了又删。不求助——被训练成不能需要。手机扣过去——所有情绪处理的缩影：感受到了压下去不让人看见。最极端压力下进入高效状态做完所有事，然后在安全地方（比如行进中的车里）一个人坐着什么都不做。 |
| 前史 | 大学时期加入某渠道过程留白。没有做过长期任务，缅北园区是第一个超过三个月的任务。习惯：每到一个新地方先找最近出口在哪里——不是逃跑是让自己安心的方式。上线"陈工"，男性，比她大十岁，不完全是职业关系的信任但从来没越过某条线。 |

## 老鸨子（NPC状态·盘总线）

盘总线中：她在被带去办公楼的路上经过他所在区域，她注意到他对方没有注意她。视觉上的"第一个人"不是交互上的。之后几乎无交叉。
